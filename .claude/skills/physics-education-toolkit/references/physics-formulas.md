# 주요 물리 공식 레퍼런스

## 역학 (Mechanics)

### 운동학
| 공식 | 설명 | 단위 |
|------|------|------|
| $v = v_0 + at$ | 등가속도 속도 | m/s |
| $x = v_0 t + \frac{1}{2}at^2$ | 등가속도 변위 | m |
| $v^2 = v_0^2 + 2ax$ | 속도-변위 관계 | m²/s² |

### 뉴턴 법칙
| 공식 | 설명 |
|------|------|
| $\vec{F} = m\vec{a}$ | 뉴턴 제2법칙 |
| $\vec{F}_{12} = -\vec{F}_{21}$ | 뉴턴 제3법칙 |
| $f \leq \mu N$ | 마찰력 |

### 회전 운동
| 공식 | 설명 |
|------|------|
| $\tau = r \times F = rF\sin\theta$ | 토크 |
| $\tau = I\alpha$ | 회전 운동 방정식 |
| $L = I\omega$ | 각운동량 |
| $I = \sum m_i r_i^2$ | 관성 모멘트 |

### 에너지
| 공식 | 설명 |
|------|------|
| $KE = \frac{1}{2}mv^2$ | 운동 에너지 |
| $PE = mgh$ | 중력 퍼텐셜 에너지 |
| $PE = \frac{1}{2}kx^2$ | 탄성 퍼텐셜 에너지 |
| $W = F \cdot d = Fd\cos\theta$ | 일 |

## 전자기학 (Electromagnetism)

### 정전기
| 공식 | 설명 |
|------|------|
| $F = k\frac{q_1 q_2}{r^2}$ | 쿨롱 법칙 |
| $E = k\frac{q}{r^2}$ | 점전하 전기장 |
| $V = k\frac{q}{r}$ | 점전하 전위 |
| $U = k\frac{q_1 q_2}{r}$ | 전기 퍼텐셜 에너지 |

### 회로
| 공식 | 설명 |
|------|------|
| $V = IR$ | 옴의 법칙 |
| $P = IV = I^2R = \frac{V^2}{R}$ | 전력 |
| $R_{직렬} = R_1 + R_2 + ...$ | 직렬 저항 |
| $\frac{1}{R_{병렬}} = \frac{1}{R_1} + \frac{1}{R_2} + ...$ | 병렬 저항 |

### 자기
| 공식 | 설명 |
|------|------|
| $F = qvB\sin\theta$ | 로렌츠 힘 |
| $F = BIL$ | 전류가 받는 힘 |
| $\Phi = BA\cos\theta$ | 자기 선속 |
| $\varepsilon = -\frac{d\Phi}{dt}$ | 패러데이 법칙 |

---

## 파동 (Waves)

| 공식 | 설명 |
|------|------|
| $v = f\lambda$ | 파동 속도 |
| $y = A\sin(kx - \omega t)$ | 파동 방정식 |
| $k = \frac{2\pi}{\lambda}$ | 파수 |
| $\omega = 2\pi f$ | 각진동수 |
| $T = \frac{1}{f}$ | 주기 |

---

## 열역학 (Thermodynamics)

| 공식 | 설명 |
|------|------|
| $PV = nRT$ | 이상 기체 방정식 |
| $Q = mc\Delta T$ | 열량 |
| $\Delta U = Q - W$ | 열역학 제1법칙 |
| $\eta = 1 - \frac{T_C}{T_H}$ | 카르노 효율 |

## 현대 물리 (Modern Physics)

### 양자역학
| 공식 | 설명 |

| $E_n = -\frac{13.6}{n^2}$ eV | 수소 에너지 준위 |

### 상대성 이론
| 공식 | 설명 |
|------|------|
